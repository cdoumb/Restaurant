
Site vitrine moderne et responsive pour un restaurant sénégalais fictif situé à Dakar.  
Le projet met en avant la gastronomie locale à travers une interface élégante, chaleureuse et intuitive.

Le baobab d'or est un site web one-page conçu pour présenter :
- l’identité du restaurant,
- ses spécialités culinaires,
- une galerie de plats,
- des témoignages clients,
- un formulaire de réservation,
- et la localisation via Google Maps.
  
 Objectif : proposer une expérience utilisateur agréable tout en valorisant la cuisine sénégalaise.
 
Technologies utilisées
- HTML — structure sémantique du site  
- CSS — design, animations et responsive design  
- Goggle map — localisation du restaurant
